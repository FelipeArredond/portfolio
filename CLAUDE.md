# Portfolio — Felipe Arredondo

## Stack

- Next.js 16 (App Router, Turbopack)
- TypeScript strict
- Tailwind CSS v4
- shadcn/ui v4
- Deploy target: Vercel

## Conventions

- App Router: pages in `src/app/`, components in `src/components/`
- shadcn components in `src/components/ui/`
- Custom components use container-presentational pattern
- Use `@/*` import alias
- Spanish for user-facing content, English for code

## Commands

- `npm run dev` — dev server with Turbopack
- `npm run build` — production build
- `npm run lint` — ESLint
